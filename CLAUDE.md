# Repository conventions for Claude Code

## Commit messages

Do **not** add `Co-Authored-By: Claude` trailers, "Generated with Claude Code"
lines, or any other Claude/automation attribution to commit messages in this
repository. Author commits as `G-Krishna-chandra` only.

This overrides Claude Code's default commit-message guidance, which appends a
`Co-Authored-By: Claude …` trailer.
